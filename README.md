# C++ Beginner Projects

This repository contains beginner-level C++ programs created while learning programming fundamentals and preparing for open-source contributions.

## Projects Included

1. Calculator
A simple console calculator that performs:
- Addition
- Subtraction
- Multiplication
- Division

2. Number Guessing Game
A game where the computer generates a random number between 1 and 100 and the user tries to guess it with hints.

3. Prime Checker
A program to check whether a given number is prime or not.

4. Palindrome Checker
A program to check wheter the input string is a palindrome.

## Language Used
C++

## Purpose
These programs are part of my practice while learning programming, data structures, and preparing for open source programs.
